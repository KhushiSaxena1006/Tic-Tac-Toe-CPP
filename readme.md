# 🎮 Tic Tac Toe Game (C++)

A simple yet interactive **Command-Line Tic Tac Toe** game developed in **C++**. This project simulates the classic two-player Tic Tac Toe game in the terminal, allowing players to compete by taking alternate turns until one player wins or the game ends in a draw.

This project was created to strengthen core C++ programming concepts such as functions, loops, conditional statements, 2D-arrays, user input handling, and game logic implementation.

---

## 📌 Features

-  Two-player gameplay
-  Alternate turns between Player X and Player O
-  Winner detection across rows, columns, and diagonals
-  Draw detection when no winning moves are possible
-  Play Again option after each game
-  Interactive command-line interface
-  Input validation to ensure valid moves

## 🎮 Game Rules

- The game is played between **two players**.
- Player 1 uses **X** and Player 2 uses **O**.
- Players take turns choosing an empty position on the board.
- The first player to align **three symbols** horizontally, vertically, or diagonally wins.
- If all positions are filled without a winner, the game ends in a draw.
